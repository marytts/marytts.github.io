MARY TTS 4.1.1: Release notes

Download MARY TTS 4.1.1

This is a bugfix release.

Bugfixes

  • Fixed a critical bug for Windows users who got an “OutOfMemoryError” when trying to start the server with unit selection voices
    • #314 Memory mapping causes OutOfMemoryError if -Xmx is set too high
  • Other small bug fixes – see http://mary.opendfki.de/query?status=closed&group=resolution&milestone=4.1.1

New voices

  • Added Telugu HMM-based voice “cmu-nk-hsmm”.
