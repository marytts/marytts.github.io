Mary 3.0.1 released

First bugfix release

Get MARY 3.0.1 from the download page.

A number of bugs have been fixed which have occurred after the initial release. Most relevant:

  • Audio can now be saved with proper audio headers (ticket 32)
  • Mary GUI client is now accessible via the keyboard (ticket 3)

An update is recommended.
